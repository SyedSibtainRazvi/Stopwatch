# Stopwatch

## Description:
A simple responsive and efficient **Stopwatch** that has _start_, _pause_ and _reset_ buttons, made using beautiful **Vanilla Js**.

### Technologies Used:
1) HTML.
2) CSS.
3) Vanilla JavaScript.
***

#### What Next?
